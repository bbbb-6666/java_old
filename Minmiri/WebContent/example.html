DAY1 : 표, 목록 만들기

○ 표를 만드는 방법
  - 표는 <,> 기호와 td, tr, th를 이용해 만든다.
     맨 윗줄에는 괄호 안에 table 넣어주고 끝날때는 /table라고 명시해준다.
     tr로 행을 만들고 td로 셀을 표현한다.
     colspand은 가로 칸 즉, 열을 합치는 기능이고 rowspan은 세로 칸, 즉 행을 합치는 기능이다.
     표의 제목 같은 기능을 하는 부분은 th로 표현해주면 된다.

○ 목록 만드는 법
   - 목록을 만드는 법에는 세가지 방법이 있다.
      1. ul, li태그는 순서 없는 목록을 만드는데 사용된다.
      → 예시
  • 일반전화 : (국번전화) 1330
  • 휴대전화 : 064-1330
      2. 순서가 있는 목록은 ol, li태그를 통해 만드는데, ol 태그의 괄호 안에 type="속성값" start="시작순서"를 적어주면 좀 더 용이하게 순서를 정리할 수 있다.

    3. 카약체험
      3. dl, dt, dd 태그는 설명 목록을 만들 때 사용한다.
        dl은 목록을 만들고, dt는 제목을, dd는 설명을 표시한다.

DAY2 : 이미지맵, 앵커기능


○ 이미지 맵
   - 정의 : 하나의 이미지에 여러 개의 링크를 거는 것
   - 사용법 :
img src="이미지명.gif" alt="이미지가 출력이 안될시 뜨는 문구" usemap="#이미지맵이름" border="0"
map name="이미지맵이름"
area shape="링크영역모양" coords
="사각형왼쪽 위 좌표, 사각형왼쪽 아래 좌표,사각형 오른쪽 위 좌표, 사각형 오른쪽 아래 좌표" href="연결페이지" target="링크한 내용이 표시될 위치"
/map
○ 앵커기능
   - 정의 : 현재 페이지에서 원하는 곳으로 이동하게 해주는 기능
   - 사용법 :
      a href="#위치고유이름" 글자내용 /a 라고 써놓고
      이동했으면 하는 위치에 id="위치고유이름" 태그를 걸어놓으면 이동한다.
